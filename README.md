Dra.Jane Doe| Smile Design and Aesthetic Rehabilitation Clinic 

Website Project Purpose and Objective

The website is a responsive bilingual dental landing page for a specific dentist specializing in smile design and aesthetic rehabilitation.
The intended aim of the website is to aid patient learn about premium dental services, comprehend treatment process, request online evaluation and contact with the doctor before traveling.
Target user patient that are english and spanish speaker patient that are looking for:

 - smile design
 - veneers
 - Aesthetic Rehabilitation
 -Crowns and Implants
 -Dental tourism support

Based on Feedback of Previous deliver Landing Page, improvement were made to this new project:

-functional scope - now include more than basic informational section like online form, treatment option, FAQ, before and after, WhatsApps and Demo upload file.

-Interaction and Engagement - features to JS were added:
  -  English/Spanish Language Tool
  -  FAQ Accordion
  -  Treatment Plan Selection
  -  Navbar Scroll Effect
  -  Form Confirmation Message 
  -  Demo File Upload Message

- UI/UX - design enhance with premium color palette. better spacing, typography hierarchy, consistent cards, more personal focused brand based on a doctor.

-Responsiveness - Bootstrap  and custom css media queries to enhance adaptation across devices.

- Prototyping tool was planned using Mockup, to structure layout, section, color and user flow in order to code.

-Tech employed:
    - HTML5
    - CSS3
    - JavaScript

    -   Responsive Testing
        - Tested on different screen sizes  like desktops, tablet, mobile and check. layout adapts employed in bootstrap grid, navigation collapses, buttons are friendly, section stack properly.
    - Form testing assure proper data submission like:
        -Required field validation, receives submission, confirmation message display, email notification receive,file upload demo and dental record link.

    - Limitations  like file upload demo, real medical data storage secure backed system and real whatsApp number.Also for this project fictitious doctor was create and image obtain online with free use right.
